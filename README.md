# Sparse-LTI-system-realization

We find sparse LTI system realizations by solving a system of multivariate polynomial equations.
These are code to reproduce the 3 numerical examples in "Sparse Dynamic Realizations of Linear Controllers". You need to setup CVX on your computer to run example 1. I recommend using MOSEK in CVX.

An improved version of the code which allows users to change support and LTI system will be made available in the future
